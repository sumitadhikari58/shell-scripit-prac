🐚 Shell Script Practice Repository

A structured learning repository for mastering Linux Bash scripting and automation.

This repository documents my journey learning Shell scripting on Ubuntu/Linux.
It contains organized scripts, exercises, and automation examples covering topics such as loops, functions, cron jobs, file handling, logging, and backups.

The goal is to build a strong understanding of Bash scripting fundamentals and real-world automation tasks.
